# Zebra block

## Installation

1. **Install** zebra-block:
```
  npm install zebra-block
```

2. **Import** component into your project
```
  import { Zebra } from 'zebra-block';
```

3. **Usage**.
```
  <Zebra json={json} />
  <Zebra>
    Lorem ipsum...
  </Zebra>
```
